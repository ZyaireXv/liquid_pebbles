# Changelog

本文件记录 `liquid_pebbles` 的版本变更。

## 0.0.3

文档修正版本：

- 将 README 中的演示动图改为 GitHub Raw 绝对地址，避免 `pub.dev` 无法渲染相对路径图片

## 0.0.2

文档更新版本：

- 更新 README 的安装说明，改为发布后的 `pub.dev` 接入方式
- 为 README 增加真实运行录屏，方便首次浏览时直接判断组件效果

## 0.0.1

初始版本，完成基础可用能力：

- 提供 `LiquidPebbles` 组件，用于展示可游走、可蠕变的鹅卵石卡片
- 支持每个 Pebble 自定义内容、尺寸、初始锚点和是否参与位置游走
- 支持位置游走与轮廓蠕变分开控制速度
- 支持控制是否允许 Pebble 之间重叠
- 支持通过 `minCardGap` 控制非重叠模式下的最小间距
- 使用连续曲线裁剪轮廓，避免出现明显直线边和规则圆角矩形感
- 提供 `example` 示例工程，包含允许重叠与禁止重叠两种展示模式
- 补充基础测试、中文 README、MIT 开源协议和仓库初始化配置
